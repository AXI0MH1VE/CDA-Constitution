## Attribution Guidelines

### Required Attribution

When using CDA-v1.0, you **must**:

1. Include link to constitution repository
2. Credit author (Alexis M. Adams / AxiomHive)
3. Specify version (v1.0.0)
4. Note license (CC BY-SA 4.0 for documentation)

### Optional Attribution

You **may** also:

- Link to specific articles/sections
- Reference implementation repository
- Include compliance badge
- Link to supporting papers

### Prohibited

You **must not**:

- Claim authorship of the constitution
- Remove attribution when redistributing
- Use constitution text without proper citation
- Modify attribution requirements
